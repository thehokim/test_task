# JSONPlaceholder API Test Framework

Automated test framework for JSONPlaceholder REST API using Python, pytest, requests, Pydantic and Allure.

## Features

- Full test coverage for Posts, Users, Comments endpoints
- Schema validation using Pydantic models
- Positive / negative / regression / smoke test groups
- Parametrized tests
- Retry mechanism for unstable requests
- Detailed logging
- Allure reports generation
- Parallel test execution
- CI/CD integration (GitHub Actions)
- Configurable via environment variables
